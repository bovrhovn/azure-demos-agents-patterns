# Source Code

This directory contains the source code for Azure Demos — Agents & Patterns.

## Structure

Organize code by agent pattern or feature area. Each sub-folder should contain a focused implementation with its own documentation.

```
src/
├── <pattern-name>/   # One folder per agent pattern or demo
│   ├── README.md     # Pattern-specific documentation
│   └── ...           # Implementation files
```

## Guidelines

- Keep each agent pattern in its own sub-folder.
- Include a `README.md` in every sub-folder explaining the pattern, its purpose, and how to run it.
- Follow the coding conventions described in the [docs](../docs/README.md).
